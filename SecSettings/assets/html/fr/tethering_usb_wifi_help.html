Point d'accès mobile

Transformez votre téléphone en point d'accès Wi-Fi portable pour que vous puissiez partager votre connexion de données mobiles avec d'autres appareils.
Pour utiliser les fonctions de point d'accès de vos téléphones, vous devez ajouter le service Smartphone Mobile HotSpot à votre forfait.
Pour activer Smartphone Mobile HotSpot, appuyez sur “Point d'accès mobile” sous “Paramètres >> Plus de réseaux >> Modem et point d'accès mobile”.
Pour configurer Smartphone Mobile HotSpot, appuyez sur “Point d'accès mobile”, puis sur “Configurer”.
Vous pouvez définir le nombre d'appareils pouvant utiliser la connexion des données mobiles dans “Configurer >> Afficher les options avancées >> Connexions maximales”.

Modem USB

Utilisez le modem pour partager la connexion des données mobiles de votre téléphone avec un ordinateur à l'aide d'un câble USB.
Pour utiliser les fonctions de modem de vos téléphones, vous devez ajouter le service Smartphone Mobile HotSpot à votre forfait.
Pour connecter votre téléphone, reliez-le à un ordinateur à l'aide d'un câble USB. Sur votre téléphone, sélectionnez ensuite “Modem USB” dans “Paramètres >> Plus de réseaux >> Modem et point d'accès mobile”.
Le modem fonctionne sous Windows®7, Windows Vista® et Linux. Pour plus d'informations sur l'utilisation du modem avec les systèmes d'exploitation, rendez-vous à l'adresse suivante : www.android.com/tether.
Sous Windows XP, il sera peut-être nécessaire d'installer un pilote USB sur votre ordinateur avant de connecter votre téléphone. Pour installer le pilote, saisissez l'URL suivante dans le navigateur de votre ordinateur.
<http://www.samsung.com/support/>
Lorsque vous connectez votre téléphone à un ordinateur à l'aide d'un câble USB, vous avez le choix entre partager la connexion des données mobiles via le modem OU partager des fichiers. Vous ne pouvez pas faire les deux. N'activez pas le modem si vous souhaitez utiliser votre ordinateur pour accéder à la carte mémoire de votre téléphone.
